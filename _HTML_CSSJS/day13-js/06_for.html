반복문 : for, while, do~while => Java와 동일한 문법

  • for : 반복횟수가 명확할 때 사용 for(초기값; 조건식; 증감식){코드...;}
  • while, do~while : 반복 횟수가 명확하지 않을 경우 while(조건식) {실행문...;} do{실행문...}while(조건식);
  • 모든 조건은 참(true)일 때 실행
  • 조건이 계속 true라면 무한 루프에 빠질 수 있음. 조건문 안에서 조건값이 false로 바뀔 수 있는 구문이 반드시 있어야 함.
  • break; 통해 빠져 나갈 수 있음.
