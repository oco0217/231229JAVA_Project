form 태그를 통해 데이터 받아오기

  • input, textarea, select, html element 속성, 텍스트 데이터를 받아오기
  • input tag는 value 속성으로 값이 맵핑 되어있어서 value로 값을 가져올 수 있음.
  • select => option / radio / checkbox value -값을 별도로 추가해주어야 선택한 값을 가져올 수 있음.
  • value / innerText 값은 모두 string 처리된다.
  • string처리된 value 값을 숫자로 사용하기 위해서는 변환이 필요하다
