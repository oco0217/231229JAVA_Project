객체(Object)

  • object : 데이터의 구조를 상세 특징으로 표현하려는 형식
  • const로 선언하는것이 일반적 : 객체의 구조를 지속성있게 유지하고, 확보하기 위함.
  • { 속셩명 : 값, 속성명 : 값, 속성명 : 값, ... }
  • 속성명 : key, 값 : value
  • 같은 object 내의 같은 속성명은 불가능 (key 중복 불가능)
  • 객체 내부에서 속성 값으로 사용할 수 있는 데이터의 제한이 없다.
  • 객체의 속성 값으로 익명함수를 정의 => 메서드라고 부름 메서드의 이름은 속성명이 됨.
