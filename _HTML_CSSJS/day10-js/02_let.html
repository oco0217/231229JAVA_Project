자바스크립트의 변수 사용

  • 값을 담고 있는 메모리 공간의 이름
  • java => 자료형 변수명;
  • 자바스크립트는 변수는 선언하는 역할만 한다.(자료형이 없음)
  • var, let(ES6 var는 권장하지 않음), const
  • let 자바에서 사용되는 지역변수의 개념
  • var name = 'hong'; var name = 'kim'; => 가능
  • let 중복선언 안됨. 재할당은 가능 let name = 'hong'; name = 'kim';
  • const: 재선언 불가능. 재할당 안됨 const name = 'hong'; name = 'kim' => 불가능
  • 스코프(scope:사용범위){ } 안에서만 사용가능 스코프를 벗어나면 재선언 가능
  • const : 고정된 값을 사용하는 규칙, 참조, 객체, element 정의시 사용
  1. 기본자료형 : 값이 메모리의 주소값을 가르키는 곳에 직접 할당이 되어있는 값 number, boolean, string(참조자료형이지만 편의상 기본자료형과 같이 사용)
  2. 참조자료형 : 값이 저장되어 있는 주소를 할당 string, array, object, function, null, undifind
  3. == 같다(값만 비교) != 같지않다 ===같다(값과 타입이 같은지 비교) !==같지않다(값과 타입이 같지않은지 비교)
